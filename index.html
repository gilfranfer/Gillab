Okulus

Do you need to track the outcome of your groups reunions?
Okulus is a real time web application that helps to track the reports created for group reunions. Admin can create groups, members, and other elements. Users will be able to create reports for the groups assigned to them. The data from the reports is used to create charts and tables for performance analysis. The system includes notifications and chat
